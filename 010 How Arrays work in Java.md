# Primitive and Non-Primitive Data Types in Java

Primitive Data Types : 
- int
- char
- long
- short
- byte
- boolean
- float
- double
- These data types goes into the Stack memory.

Non-Primitive Data Types : 
- Arrays
- Strings
- 
- These data types goes into the Heap Memory and the variable of these data types are called Reference Variables which do not keep values themselves but point to the values in memory.
